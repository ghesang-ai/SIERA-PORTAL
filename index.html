Tips Hemat Token
🗄️
Prompt Caching
System prompt per modul di-cache — tidak dikirim ulang setiap request ke Claude API.
Hemat ~80% token
🔀
Lazy Load Modul
Satu request hanya load konteks 1 modul — tidak buang token untuk 3 modul sekaligus.
3x lebih efisien
⚡
Haiku untuk Routing
Classifier dan task ringan pakai Haiku (~5x lebih murah). Sonnet hanya untuk analisis berat.
5x lebih murah
📦
Preprocess Input
Agregasi CSV sebelum dikirim, resize image ke 512px, strip data yang tidak relevan.
Kurangi input token
🎯
max_tokens per Modul
Set batas 256–512 token per response. Cegah output panjang yang tidak dibutuhkan.
256–512 token
🗂️
JSON Output Schema
Paksa output ke format JSON ringkas. Mudah di-parse tanpa perlu re-proses dengan AI lagi.
Structured output
